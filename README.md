## 🎯 产品定位

**产品名称**: RPI关系人格指数
**核心功能**: 30道犀利毒舌题目 → 16种人格标签 → 毒舌解读 → 社交分享

## 🎨 设计风格

- **主题**: 赛博朋克霓虹
- **配色**: 深色背景 + 渐变光效（#FF6B6B、#4ECDC4、#1A1A2E）
- **动画**: 粒子背景、脉冲光效、平滑过渡

## 📊 16种人格类型

4个维度各4种组合，共16种人格：

- **A维度（进攻性）**: A-主动型、P-被动型
- **B维度（边界感）**: I-清晰型、B-模糊型
- **C维度（情感模式）**: F-感性型、C-理性型
- **D维度（沟通风格）**: D-直接型、E-迂回型

## 🚀 使用方法

### 微信小程序使用

1. 将 `rpi/` 文件夹复制到小程序项目的 `pages/` 目录下
2. 将 `rpi-data.js` 和 `rpi-calculator.js` 复制到 `utils/` 目录下
3. 在 `app.json` 中添加页面路由：

```json
{
  "pages": [
    "pages/rpi/start/start",
    "pages/rpi/quiz/quiz",
    "pages/rpi/result/result"
  ]
}
```

4. 在首页或互动页添加入口跳转：

```javascript
wx.navigateTo({
  url: '/pages/rpi/start/start'
})
```
