# GitHub同步操作指南

## 请在终端中复制以下命令并执行（按顺序一条条执行）：

```bash
# 第1步：进入目录
cd ~/Desktop/AI合集/测试

# 第2步：初始化Git仓库
git init

# 第3步：配置远程仓库
git remote add origin git@github.com:lingxiyin123-cpu/-2.git

# 第4步：添加文件
git add .

# 第5步：提交
git commit -m "添加RPI关系人格指数测试H5应用"

# 第6步：推送
git push -u origin master
```

---

## 推送成功后，启用GitHub Pages：

### 1. 访问设置页面
https://github.com/lingxiyin123-cpu/-2/settings/pages

### 2. 配置GitHub Pages
在"GitHub Pages"部分找到以下选项并设置：

- **Source**: 选择 `Deploy from a branch`
- **Branch**: 选择 `master`
- **Folder**: 选择 `/ (root)`
- 点击 **Save** 按钮

### 3. 等待部署
等待1-2分钟，GitHub会自动构建并部署

### 4. 访问网站
https://lingxiyin123-cpu.github.io/-2/

---

## 常见问题

### 如果提示"Permission denied (publickey)：
说明SSH密钥未配置。需要：
1. 生成SSH密钥：`ssh-keygen -t rsa -b 4096 -C "lingxiyin123@gmail.com"`
2. 复制公钥：`cat ~/.ssh/id_rsa.pub`
3. 添加到GitHub：https://github.com/settings/keys

### 如果提示"remote origin already exists"：
执行：`git remote set-url origin git@github.com:lingxiyin123-cpu/-2.git`

### 如果推送失败但文件已存在：
执行：`git push -u origin master --force`
